---
title: Types d’actions
slug: /reference/actions
---

Les actions disponibles sont les suivantes.

- `move` déplace un objet dynamique de `from` vers `to`.
- `arrow` dessine une flèche animée.
- `parallel` exécute plusieurs actions simultanément.
- `loading` affiche un état de chargement.
- `set_content` remplace le contenu d’un nœud.
- `comment` affiche une annotation textuelle.
- `highlight` met en évidence un nœud ou une connexion.
