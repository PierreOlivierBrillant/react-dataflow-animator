React DataFlow Animator compile une spécification JSON en animation déterministe et navigable de flux de données.

## Aperçu

Vous décrivez des objets statiques, des objets dynamiques (qui se déplacent entre les objets entre les objets statiques) et une suite d’actions.
Le moteur place les noeuds, trace les trajets et déroule la timeline sans coordonnées manuelles.

<DataFlowPlayer theme="auto" spec={demosById.intro.spec} />

## Principes

- Le temps est l’unique source de vérité, donc le seek et la navigation par étapes restent fiables.
- La disposition est calculée automatiquement en linéaire ou en circulaire.
- Le composant est SSR-safe et utilisable directement dans Docusaurus.
